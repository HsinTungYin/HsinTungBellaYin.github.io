Final Project Proposal

Description

I will create a Typing Catch Game using JavaScript. In this game, letters start falling from the top of the screen, and the player needs to type the corresponding letter to catch each one and prevent it from hitting the bottom. As the game progresses, the falling speed increases, making it more challenging. The game keeps track of the player's score, and missing too many letters results in a game over.

What I Don't Yet Know / Expect to Be Difficult

  • How to make the game visually appealing with smooth animations
  • How to track the score and display a game-over message after multiple misses

What I Already Know

  • Detecting keyboard input with event listeners
  • Moving objects on screen using a for loop
